##  Quick Reference

### Key Concepts

**Matrix Operations:**
- Determinant: tells if a matrix is invertible
- Matrix multiplication: row × column (order matters!)
- Inverse matrix: $A^{-1}$ where $AA^{-1} = I$

**Modular Arithmetic:**
- $a \equiv b \pmod{m}$
- Used in all modern encryption
